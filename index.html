3 Col Portfolio

Dry Bean Classification

To improve packing efficiency, U.S. farmers use high-speed cameras on production lines to capture individual bean features. This project aims to help the manufacturer develop a high-accuracy multi-class classifier to determine which beans belong in specific packages on the conveyor belt.

Taiwanese House Price Prediciton

As part of Taiwan's younger generation, planning to purchase a home is common when starting a family. This project aims to analyze how real-world factors influence house prices and predict prices based on a home's specifications, enhancing understanding of the housing market.

Fashion Industry Image Classification

The goal of this project is to explore how a multi-layer perceptron (MLP) can identify clothing images from RGB pixel values, transforming nonlinear patterns into meaningful, human-recognizable clothing objects.
